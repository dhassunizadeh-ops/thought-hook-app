An interactive prototype demonstrating core UX principles from Steve Krug's *Don't Make Me Think*, Don Norman's *The Design of Everyday Things*, and Nir Eyal's *Hooked*.

## What It Teaches

- **Krug (Usability)**: One obvious primary action (+ New button), clear interface hierarchy, minimal cognitive load
- **Norman (Design Principles)**: Signifiers and mapping (intuitive icons), immediate feedback (toast notifications), visible system status (streak counter)
- **Eyal (Hook Model)**: Complete engagement loop with trigger (visible streak), action (add note), variable reward (toast + streak increment), and investment (saved notes build user commitment)
